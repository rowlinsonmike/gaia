## Overview

GAIA helps manage the lifecycle of terraform deployments stored in AWS CodeCommit.

## Key Features 

- 🧹 terraform linting
- 🐛 static code analysis
- 🗺️ terraform plan capture
- 📊 terraform state visualization
- ✅ terraform apply capture
- 📍 user action tracking
